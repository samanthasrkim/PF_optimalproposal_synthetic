# Particle filter implementation with optimal proposal

This jupyter notebook contains the code for a particle filter implemented with an optimal proposal in synthetic experiments.

## Requisites

Jupyter notebook

## Files

* PFOP_synthetic_release130623                  Contains the main code
* function_compute_def130623                    Contains the function to compute subsidence
* input_files/reservoir_setup_groningen         Contains  reservoir properties to mimic the Groningen gas field
* input_files/nuclei_position_groningen         Contains the location of the nuclei of strain to create a disk-shaped reservoir (Geerstma et al. 1973)
* input_files/meanpressure_in_time.txt          Contains mean pressure values (in bar) of the reservoir (NAM report Reservoir Pressure and Subsidence Groningen Field update for Production Profile 2021)
